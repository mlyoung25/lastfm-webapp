Last.fm+

Connect your Last.fm account to use custom features the main site doesn’t offer: export recent tracks, top artists, and any API method you want to add.

Get an API key at last.fm/api/account/create and set LASTFM_API_KEY and LASTFM_API_SECRET in .env.
